# 僵尸危机：方块头 · 经典复刻版

一个单文件 HTML5 Canvas 小游戏，致敬经典 Boxhead 玩法。打开页面即可游玩，无需安装依赖或启动服务。

## 项目简介

玩家将在俯视角荒野地图中抵挡一波波僵尸和特殊敌人的进攻，通过连续击杀提升连击倍率，并逐步解锁更强武器。游戏内置键盘、鼠标和触屏操作，适合直接在浏览器中体验。

## 主要特性

- 单文件实现，包含 HTML、CSS 和 JavaScript
- Canvas 渲染的俯视角动作射击玩法
- 菜单、暂停、结算和操作说明界面
- 键盘经典模式、鼠标点击模式和移动端触屏摇杆
- 波次刷怪、连击倍率、武器解锁和掉落补给
- Web Audio 实时合成音效，无需额外音频资源

## 如何运行

直接用浏览器打开：

```text
僵尸危机-方块头-经典复刻版_5.html
```

推荐使用 Chrome、Edge、Firefox 等现代浏览器。

## 操作方式

- 移动：`W` `A` `S` `D` 或方向键
- 射击：`空格` 或 `J`
- 切换武器：`Q` / `E` 或数字键 `1` 到 `6`
- 暂停：`P` 或 `Esc`
- 鼠标模式：左键点击地面移动，角色朝向鼠标方向瞄准
- 触屏模式：左摇杆移动，右摇杆瞄准射击
