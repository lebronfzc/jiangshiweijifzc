▍操作说明

键盘经典模式：WASD 或方向键移动，朝面对方向射击
鼠标射击模式：左键点击/按住朝指针方向攻击；右键点击移动到该点、按住持续跟随；也可用 WASD 移动
射击：左键 / 空格 / J（可按住连射）
切换武器：鼠标滚轮 / Q E / 数字键 1－9 ｜ 暂停：P / Esc

▍游戏规则

抵御从四周涌来的方块头僵尸与会喷火球的红恶魔。每波一只 BOSS，共五级递进： 壮汉巨尸（召唤）→ 掠行猎尸（连环冲锋+毒吐）→ 碎地暴尸（召唤/冲锋/震地）→ 灾祸巨像（再加全向弹幕）→ 僵尸博士（巨大体型·全技能·毒池·残血狂暴）。 脚下红圈/绿圈是技能预警，快躲开！击破 BOSS 必掉补给。 连续击杀提升连击倍数， 倍数达到 ×3 / ×5 / ×8 / ×11 / ×13 / ×15 / ×17 / ×20 依次解锁： 马格南、乌兹冲锋枪、霰弹枪、加特林、手雷、油桶、喷火器、火箭筒。 受到伤害连击减半。敌人偶尔掉落医疗箱与弹药箱。地上的油桶射击即可引爆，还会连环殉爆！
